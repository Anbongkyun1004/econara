환경 데이터 분석 수업 지도안

수업 개요 1차시 2차시 3차시 평가 계획

수업 개요

이 수업은 고등학교 1학년 학생들이 환경 데이터의 중요성을 깨닫고, 데이터를 직접 정리, 시각화, 해석하는 과정을 통해 과학적 탐구 역량과 환경 문제에 대한 비판적 사고력을 함양하는 것을 목표로 합니다.

핵심 정보

  • 과목: 통합과학 (환경 영역)
  • 단원: 과학 탐구 (환경 데이터 실습)
  • 대상: 고등학교 1학년
  • 차시: 총 3차시 (각 50분)

수업 목표

  1. 환경 현상에서 데이터를 수집하고 기록하는 방법을 이해한다.
  2. 환경 데이터를 효과적으로 정리하고 시각화할 수 있다.
  3. 환경 데이터를 통해 경향성을 파악하고 합리적인 결론을 도출할 수 있다.
  4. 데이터 분석의 한계와 환경 윤리적 측면을 인식하고 비판적으로 사고한다.

수업 준비물

교사

  • 빔 프로젝터, 컴퓨터, 판서 도구
  • 활동지, 예시 데이터 세트

학생

  • 필기도구, 스마트폰 또는 태블릿
  • 개인 노트북 (선택)

1차시: 환경 데이터, 어디에 숨어있을까?

"환경 탐정 되기"

1차시는 학생들이 '환경 데이터'라는 개념에 익숙해지는 시간입니다. 일상 속에서 환경 데이터를 발견하는 활동과 토론을 통해 데이터 분석의 필요성을 스스로 깨닫게 합니다. '파인만 기법'을 활용하여 어려운 개념을 쉽게 설명하는 연습도 함께 진행됩니다.

수업 흐름

▼
도입 (10분)

'환경 데이터' 경험에 대해 질문하고, 파인만 기법을 소개하며 학습 동기를 유발합니다.

💡 파인만 기법 예시: "환경 데이터는 5살 동생에게 설명하듯 가장 쉽게 설명하는 거예요! '오늘 미세먼지 나쁨!' 처럼요."
전개 (30분)

활동 1: '나만의 환경 데이터' 찾기 (짝과 함께 주변의 환경 데이터를 찾아보고 설명하기)

활동 2: '환경 문제 해결에 데이터가 중요한 이유' 토론 (모둠별로 분석의 필요성 토의 및 발표)

정리 (10분)

핵심 내용을 정리하고, 다음 차시를 예고하며 과제를 제시합니다.

과제: 주변 환경 현상에서 흥미로운 데이터 3가지 찾아오기

2차시: 환경 데이터, 어떻게 정리하고 표현할까?

"데이터를 옷 입히기"

2차시에서는 수집한 환경 데이터를 효과적으로 정리하고 표현하는 방법을 배웁니다. 표와 다양한 그래프(막대, 꺾은선)의 특징을 이해하고, 주어진 데이터를 가장 적절한 형태로 시각화하는 실습을 진행합니다. 아래에서 직접 차트를 만들어보며 각 그래프의 특징을 확인해보세요.

활동: 환경 데이터 그래프로 시각화하기 (실습)

아래는 '서울의 월별 평균 미세먼지(PM10) 농도' 가상 데이터입니다. 버튼을 클릭하여 데이터를 다양한 그래프로 표현해보세요. 어떤 그래프가 월별 비교에 더 효과적일까요? 또 어떤 그래프가 시간의 흐름에 따른 변화를 보기에 더 좋을까요?

3차시: 환경 데이터, 무엇을 말해줄까?

"환경의 속삭임 듣기"

마지막 3차시는 시각화된 데이터 속에서 의미 있는 경향성과 패턴을 찾아내는 해석 능력을 기릅니다. 또한, 데이터 분석의 한계점을 배우며 정보를 비판적으로 수용하는 태도를 함양합니다. 아래 '탐정의 질문'을 통해 데이터가 숨기고 있는 이야기를 찾아보세요.

아래 그래프는 '1880년부터 2020년까지 전 지구 연평균 기온 편차' 데이터입니다. 그래프를 보고 아래 '탐정의 질문' 버튼을 클릭하여 데이터가 말해주는 숨은 의미를 찾아보세요.

평가 계획

학생들의 학습 과정과 결과는 수행평가와 관찰 평가를 통해 종합적으로 평가합니다. 이를 통해 데이터 분석 능력뿐만 아니라 협업 능력과 환경 문제에 대한 비판적 사고력까지 다면적으로 평가하고자 합니다.

수행평가 (80%)

  • 활동지 작성 (40%): 각 차시별 활동지 참여도 및 내용 충실성
  • 모둠 활동 및 발표 (40%): 모둠 내 협력 태도, 발표의 논리성 및 명확성

관찰 평가 (20%)

  • 환경 데이터 분석 보고서: 최종 보고서의 완성도 및 논리성
  • 수업 참여 태도: 질문의 적극성 및 파인만 기법 이해도
